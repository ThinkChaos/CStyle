CStyle
======
A tool to help detect (some) deviations from [EPITA](http://www.epita.fr/)'s C99/C++ Coding Style Standard.


Installation
------------
    $ git clone https://github.com/ThinkChaos/CStyle.git


Testing
-------
While in the git repo:
    $ python -m unittest


Usage
-----
Requires Python 3.

    $ python /path/to/cstyle [files...]

Adding cstyle to your path allows to run:

    $ python -m cstyle [files...]
